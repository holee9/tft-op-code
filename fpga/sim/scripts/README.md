# Simulation Scripts

Scripts for running simulations and generating reports.

## Available Scripts

| Script | Description |
|--------|-------------|
| `run_all.sh` | Run all simulations |
| `run_unit.sh` | Run unit tests only |
| `compile.sh` | Compile RTL and testbenches |
| `wave.do` | Waveform display configuration |
