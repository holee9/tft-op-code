# FPGA Utility Scripts

Utility scripts for FPGA development and deployment.

## Available Scripts

| Script | Purpose |
|--------|---------|
| `build.sh` | Build firmware |
| `program.sh` | Program FPGA |
| `verify.sh` | Verify firmware |

## Usage

```bash
# Build firmware
./build.sh

# Program FPGA (via JTAG)
./program.sh

# Verify programmed firmware
./verify.sh
```
