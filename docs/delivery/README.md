# Delivery Documentation

Documents for delivery to external development teams.

## Directory Structure

### to-yocto-project/
Specifications for the Yocto Project Linux BSP team.

- **Linux kernel driver requirements** for SPI communication
- **Device tree configurations** for i.MX8 GPIO and SPI
- **Kernel configuration requirements**
- **Userspace interface specifications**

### to-dotnet-project/
Specifications for the .NET application team.

- **API specifications** for hardware abstraction layer
- **Data models** for measurements and configuration
- **UI requirements** and workflow specifications
- **Communication protocol** with Linux backend

## Delivery Checklist

### Yocto Project Team
- [ ] SPI driver specification
- [ ] Device tree source files
- [ ] Kernel configuration fragment
- [ ] Test application requirements

### .NET Application Team
- [ ] Hardware abstraction layer API
- [ ] Data model definitions
- [ ] UI wireframes and specifications
- [ ] Example configuration files
