# TFT Leak Reduction Project - Documentation

This directory contains all project documentation organized by category.

## Directory Structure

### panel/
Panel-specific documentation including specifications, physics analysis, and characterization data.

- **spec/**: Panel specifications and requirements
- **physics/**: Physical analysis of aSi TFT leakage behavior
- **characterization/**: Panel characterization data and measurements

### common/
Common documentation shared across subsystems.

- **spi/**: SPI protocol documentation and register definitions
- **protocol/**: Communication protocols between subsystems

### reference/
Reference documents and research papers.

- Vendor cooperation guides
- Implementation plans and execution strategies
- Academic research drafts

### design/
System design specifications and architecture documents.

- FPGA control specifications
- i.MX8 control specifications
- SPI register maps
- Control sequence flows

### delivery/
Documents for delivery to external teams.

- **to-yocto-project/**: Linux kernel and driver specifications
- **to-dotnet-project/**: Application layer interface specifications

### images/
Project images including diagrams, waveforms, and screenshots.
